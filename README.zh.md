# MediaTracker AI

[English](README.md) | [中文](README.zh.md)

一款跨平台的媒体追踪与收藏桌面应用，支持电影、电视剧、书籍、漫画、短剧和音乐。

基于 **React + Vite** (前端) 和 **Tauri/Rust** (后端) 构建。

专为速度、隐私和极致用户体验而设计。

## ✨ 主要功能

### 🔍 AI 驱动的搜索与发现
- **智能搜索**：使用自然语言查找作品（例如：“90年代的赛博朋克小说”）。
- **热门推荐**：基于当前趋势获取个性化推荐，具备智能刷新逻辑，自动排除已看过的推荐内容。
- **首屏秒开**：AI 上下文快速返回初步结果，元数据异步加载补全，无需等待。

### 📚 全面的收藏管理
- **多类型支持**：统一管理电影、电视剧、书籍、漫画、短剧和音乐专辑。
- **状态追踪**：将条目分类为“待看”、“看过”和“最爱”。
- **追更模式**：针对连载中的作品（电视剧、动漫、漫画），支持自动检查并提示最新剧集/章节更新。

### 🛠️ 高级编辑与自定义
- **手动添加**：无需搜索，直接创建自定义媒体卡片。
- **元数据编辑**：完全控制标题、导演/作者、简介、上映日期和演职人员信息。
- **自定义海报**：上传本地图片作为封面，或让应用自动从网络抓取高质量海报。

### 📊 数据洞察与统计
- **年度报告**：可视化展示您的年度观影/阅读活动，包括添加总数、最活跃月份和最爱类别。
- **搜索诊断**：查看 AI 交互和搜索提供商的详细日志（Token 消耗、延迟、API 调用耗时），方便调试与优化。

## 🚀 技术亮点

- **隐私优先**：API Key（OpenAI, Google 等）使用 **AES 加密** 存储在本地。您的数据不会上传到任何云端服务器。
- **离线友好**：支持断网查看已缓存的元数据和图片。
- **异步水合**：海报和详细信息在后台静默加载，保持界面流畅不卡顿。
- **国际化**：完整支持简体中文和英文，界面语言自动适配或手动切换。

## 🛠️ 快速开始

### 前置要求
- Node.js (v18+)
- Rust (最新稳定版)

### 开发
1. **安装依赖**:
   ```bash
   npm install
   ```
2. **启动桌面应用**:
   ```bash
   npm run tauri dev
   ```
3. **仅启动前端网页**:
   ```bash
   npm run dev
   ```

### 构建
为当前操作系统构建应用程序：
```bash
npm run tauri build
```

## 🔒 隐私与安全
- **本地存储**：所有收藏数据仅存储在您的设备上。
- **密钥加密**：敏感的 API Key 在存储前均经过加密处理。
- **无追踪**：我们不收集您的搜索历史或收藏数据。
